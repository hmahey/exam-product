Explain the concept of Responsive Design. What does the term mean?

Responsive design means a design that is adjustable according to your screen. Responsive design provides a customized viewing experience of different browser platforms. Responsive web design works through Cascading Style Sheets (CSS), using various settings to serve different style properties depending on the screen size, orientation, resolution, color capability and other characteristics of the user’s device




How does responsive design relate to web technologies?

Responsive design has a very huge relation with the web tech because Web can access through different browsers even different screen sizes like desktop, tablet, mobile devices etc. So a responsive design is very important for web.
